# 📧 Automated Email Response System

An intelligent web-based application that simulates an **Automated Email Response System**. The project provides a modern email dashboard where users can view customer emails, generate AI-inspired responses, and simulate sending replies through a clean and responsive interface.

## 🌐 Live Demo

🚀 **Live Website 1:** https://willowy-violet-jtcvwhzz.edgeone.dev/

🚀 **Live Website 2:** https://genuine-coral-skhngpvr.edgeone.dev/

> **Note:** The current deployment is a frontend demonstration. Real email reading and automatic sending require a backend (such as Node.js/Python) and integration with an email provider like Gmail API or Microsoft Graph.

---

# 📖 Overview

Customer support teams receive hundreds of emails every day. This project demonstrates how repetitive email responses can be streamlined through automation.

The system provides a Gmail-inspired dashboard where users can:

- View incoming customer emails
- Read email content
- Generate intelligent reply suggestions
- Simulate sending automatic responses
